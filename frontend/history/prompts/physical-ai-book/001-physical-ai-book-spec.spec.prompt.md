## Prompt

Intent

Create a premium, structured, Docusaurus-powered book-style documentation website for the capstone topic Physical AI, beginning with Module 1: The Robotic Nervous System (ROS 2).
The site must present a clean, book-like reading experience with a persistent sidebar, smooth scrolling, mobile-first layout, and fully MDX-editable content.

Required Pages
1. Home

Introduction to Physical AI

Overview of modules

Call-to-action for beginning the book

2. Module 1 (Book Layout)

Scrollable MDX-based book reading interface

Persistent sidebar that auto-highlights chapters

Autogenerated navigation and pagination

Exactly four chapters, detailed below

3. About

Purpose of Physical AI

Explanation of how humanoid robotics is taught

4. Contact

Simple contact form

Save form data to localStorage only (no backend)

Module 1: The Robotic Nervous System (ROS 2)

Exactly four core chapters, each written and delivered in MDX:

ROS 2 Middleware

Explain DDS, communication layers, QoS, and middleware role

Nodes, Topics, Services

Node lifecycle, publishers, subscribers, service calls, message flow

Python-to-ROS Bridging via rclpy

Writing Python nodes, timers, callbacks, executors, launch basics

URDF for Humanoids

URDF structure, joints, links, sensors, humanoid-specific modeling

Each chapter must be clearly structured, readable, and include examples, diagrams (MDX-friendly), and code blocks.

Success Criteria

Module 1 contains exactly four chapters, fully written in MDX.

Clean, premium, book-like design with persistent sidebar and smooth scroll.

Dark/light mode support.

Excellent mobile experience with responsive, polished layout.

Search enabled (Algolia or built-in).

All content editable through MDX files.

Autogenerated navigation and sidebar entries.

Fully functional Docusaurus project that can build instantly.

No backend or auth components required.

Non-Goals

Do not include:

Backend systems

Authentication or user accounts

Payments

Comment systems

Robotics simulations

Robotics code execution
